# Các thẻ thông dụng  
`h1 - h6 : heading `  
```html
        <!-- Heading -->
        <h1>Heading 1</h1>
        <h2>Heading 2</h2>
        <h3>Heading 3</h3>
        <h4>Heading 4</h4>
        <h5>Heading 5</h5>
        <h6>Heading 6</h6>
```
`p : đoạn văn  `  
```html
        <!-- paragraph -->
        <p>Đây là đoạn văn</p>
```
`img : hình ảnh`  
```html
        <!-- img -->
            <img src="src-ảnh" alt="chú-thích-nếu-ảnh-bị-lỗi">
```
`a : anchor : mỏ neo`
```html
        <!-- anchor -->
            <a href="link">hiển-thị-link-đó</a> 
```
`ul -li`
```html
<!-- ul - li -->
                <ul>
                    <li>Phần đầu</li>
                    <li>phần 2</li>
                    <li>3</li>
                    <li>4</li>
                    <li>5</li>
                </ul>
```
`table`
```html
 <!-- table -->
                <table>
                    <thead>
                        <!-- thead -->
                        <th>STT</th>
                        <th>Tên</th>
                        <th>Địa chỉ</th>
                    </thead>
                    <tbody>
                        <!-- tr table row -->
                        <!-- mỗi table row tr là 1 dòng -->
                        <tr>
                            <!-- td tuong ứng 3 thẻ th -->
                          <td>1</td>
                          <td>Thanh Tịnh</td>
                          <td>Thành Phố Hồ Chí Minh</td>  
                        </tr>
                        <tr>
                          <td>2</td>
                          <td>Kim Sejoeng</td>
                          <td>Korean</td>
                        </tr>
                    </tbody>
                </table>
```
`input`
```html
<!-- input -->
                <input type="text">
                <!-- cb -> nhiều cái -->
                <input type="checkbox">
                <!-- rdio -> chỉ 1 -->
                <!-- Cách để chọn 1 cái là đặt 3 cái cùng tên -->
                <input type="radio" name="Gender" id="">
                <input type="radio" name="Gender" id="">
                <input type="radio" name="Gender" id="">
```
`button`
```html
            <!-- button -->
                <button>Bấm vào đây, đây không phải nút</button>
```
`div`
```html
            <!-- div -->
                <div>
                    Tạo khối xung quanh
                </div>
```
